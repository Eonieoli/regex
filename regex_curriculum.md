# 1. 과정 개요 (Course Overview)

**과정 제목**: 파이썬을 활용한 정규표현식

**과정 설명**:

정규표현식(Regular Expression)은 문자열에서 특정 패턴을 찾고, 추출하고, 변환하기 위한 강력한 미니 언어입니다. 로그 분석, 데이터 정제, 입력 검증, 웹 스크래핑, 텍스트 마이닝 등 거의 모든 텍스트 처리 작업에서 정규표현식은 핵심 도구로 활용됩니다. 정규표현식을 모르면 수십 줄의 코드로 작성해야 할 작업을, 한 줄의 패턴으로 해결할 수 있는 것이 정규표현식의 힘입니다.

이 과정은 정규표현식에 대한 사전지식이 전혀 없는 학습자를 대상으로, 정규표현식의 기본 문법부터 파이썬 `re` 모듈의 활용, 고급 패턴 기법, 실전 응용, 그리고 엔진 내부 원리와 성능 최적화까지 체계적으로 다룹니다. 과정을 완료하면 복잡한 텍스트 처리 문제를 정규표현식으로 독립적으로 설계·구현·최적화할 수 있는 마스터 수준에 도달하게 됩니다.

파이썬의 `re` 모듈을 중심으로 학습하되, 정규표현식의 보편적 원리도 함께 다루어 다른 언어나 도구에서도 정규표현식을 자유롭게 활용할 수 있는 기반을 마련합니다.

**대상 학습자**: 파이썬 기본 문법(변수, 문자열, 리스트, 함수, 파일 입출력)을 이해하고 있으며, 정규표현식에 대한 사전 지식이 없는 성인 학습자

**전체 예상 학습 시간**: 약 54시간

**선수 과목/지식**:
- 파이썬 기본 문법 (변수, 자료형, 제어문, 함수, 문자열 메서드)
- 파이썬 파일 입출력 기초
- 기본적인 문자열 처리 경험 (split, join, replace 등)

### Chapter 2: 메타문자와 리터럴 매칭

- **학습 목표**:
  - 파이썬 정규표현식의 핵심 메타문자(`. ^ $ * + ? | \ ( )`)의 의미와 동작을 설명할 수 있다
  - 임의의 문자(`.`), 시작(`^`, `\A`), 끝(`$`, `\Z`) 앵커를 사용하여 위치 기반 매칭을 수행할 수 있다
  - 선택(`|`)과 이스케이프(`\`)를 활용하여 기본 패턴을 작성할 수 있다
  - raw string(`r''`)의 필요성을 이해하고 올바르게 사용할 수 있다
- **핵심 키워드**: 메타문자, 와일드카드(`.`), 앵커(`^`, `$`, `\A`, `\Z`), 선택(`|`), 이스케이프(`\`), raw string, 리터럴 매칭, 특수 이스케이프 시퀀스(`\n`, `\t`)
- **도입 개념**: `.`(임의 문자), `^`(시작 앵커), `$`(끝 앵커), `\A`(문자열 시작 절대 앵커), `\Z`(문자열 끝 절대 앵커), `|`(선택/OR), `\`(이스케이프), `()`(그룹화 기초 소개), raw string(`r''`), `\n`·`\t` 등 이스케이프 시퀀스
- **선수 지식**: Chapter 1 (정규표현식 기본 개념, 리터럴과 메타문자의 구분)
- **난이도**: ⭐⭐
- **예상 학습 시간**: 3시간
- **Section 목록**:
  - 2.1 리터럴 매칭의 원리 — 일반 문자의 매칭 동작, 대소문자 구분
  - 2.2 임의의 문자: 마침표(`.`) — 와일드카드의 동작과 제한 사항(줄바꿈 문자)
  - 2.3 위치 앵커: `^`, `$`, `\A`, `\Z` — 문자열의 시작과 끝을 지정하는 앵커, `^`/`$`와 `\A`/`\Z`의 차이(MULTILINE에서의 동작 차이 예고)
  - 2.4 선택 연산자: `|` — OR 논리를 패턴에 적용하기
  - 2.5 이스케이프와 raw string — `\`를 이용한 메타문자의 리터럴 사용, `\n`·`\t` 등 특수 이스케이프 시퀀스, 파이썬 raw string의 필요성과 활용
  - 2.6 메타문자 종합 연습 — 여러 메타문자를 조합한 기초 패턴 작성 연습

  - 단어 경계(`\b`)를 활용하여 정밀한 패턴 매칭을 수행할 수 있다
- **핵심 키워드**: 문자 클래스, 대괄호 표현, 범위 표현, 부정 문자 클래스, `\d`, `\D`, `\w`, `\W`, `\s`, `\S`, `\b`, `\B`
- **도입 개념**: 문자 클래스(`[...]`), 범위(`[a-z]`), 부정 문자 클래스(`[^...]`), `\d`, `\D`, `\w`, `\W`, `\s`, `\S`, `\b`(단어 경계), `\B`
- **선수 지식**: Chapter 2 (메타문자, 이스케이프 개념, raw string)
- **난이도**: ⭐⭐
- **예상 학습 시간**: 3시간
- **Section 목록**:
  - 3.1 문자 클래스의 기본: `[...]` — 대괄호를 이용한 문자 집합 정의, 문자 클래스 내부의 메타문자 규칙
  - 3.2 범위 표현과 부정 — `[a-z]`, `[0-9]`, `[가-힣]` 등의 범위와 `[^...]`를 이용한 제외 패턴
  - 3.3 사전 정의 문자 클래스 — `\d`, `\w`, `\s`와 그 부정형 `\D`, `\W`, `\S`의 정의와 매칭 범위
  - 3.4 단어 경계: `\b`와 `\B` — 단어 경계의 개념과 정밀 매칭에의 활용
  - 3.5 문자 클래스 조합 패턴 — 문자 클래스와 메타문자를 결합한 실용 패턴 연습

### Chapter 4: 수량자와 반복 패턴

- **학습 목표**:
  - 기본 수량자(`*`, `+`, `?`)의 동작 차이를 명확히 구분하여 사용할 수 있다
  - 범위 수량자(`{n}`, `{n,}`, `{n,m}`)를 사용하여 정확한 반복 횟수를 지정할 수 있다
  - 탐욕적(greedy) 매칭과 비탐욕적(lazy) 매칭의 차이를 이해하고, `?` 접미사로 전환할 수 있다
  - 수량자를 문자 클래스와 결합하여 실용적인 패턴을 구성할 수 있다
- **핵심 키워드**: 수량자, `*`, `+`, `?`, `{n}`, `{n,m}`, 탐욕적 매칭(Greedy), 비탐욕적 매칭(Lazy/Non-greedy), 백트래킹(개념 소개)
- **도입 개념**: `*`(0회 이상), `+`(1회 이상), `?`(0회 또는 1회), `{n}`, `{n,}`, `{n,m}`(범위 수량자), 탐욕적 매칭, 비탐욕적 매칭(`*?`, `+?`, `??`, `{n,m}?`), 백트래킹(기초 개념)
- **선수 지식**: Chapter 2 (메타문자와 기본 패턴), Chapter 3 (문자 클래스)
- **난이도**: ⭐⭐⭐
- **예상 학습 시간**: 3시간
- **Section 목록**:
  - 4.1 기본 수량자: `*`, `+`, `?` — 각 수량자의 의미, 동작 방식, 차이점
  - 4.2 범위 수량자: `{n}`, `{n,}`, `{n,m}` — 정확한 반복 횟수 지정
  - 4.3 탐욕적 매칭 vs 비탐욕적 매칭 — 두 매칭 전략의 차이, `?` 접미사를 이용한 비탐욕적 전환
  - 4.4 수량자와 문자 클래스의 결합 — `\d+`, `[a-z]{3,5}` 등 실용 패턴 구성
  - 4.5 수량자의 함정과 주의사항 — 빈 문자열 매칭(`*`), 과도한 매칭, 백트래킹 기초 개념

### Chapter 5: re 모듈 기본 함수

- **학습 목표**:
  - `re.match()`, `re.search()`, `re.fullmatch()`의 차이를 이해하고 적절히 선택하여 사용할 수 있다

### Chapter 8: 고급 그룹핑

- **학습 목표**:
  - 명명 그룹 `(?P<name>...)`으로 가독성 높은 패턴을 작성하고, 이름으로 결과를 참조할 수 있다
  - 비캡처 그룹 `(?:...)`의 용도를 이해하고, 캡처 그룹과 구분하여 사용할 수 있다
  - 패턴 내 역참조(`\1`, `(?P=name)`)를 사용하여 앞서 매칭된 내용을 재참조하는 패턴을 작성할 수 있다
  - `groupdict()`를 활용하여 명명 그룹의 결과를 딕셔너리로 추출할 수 있다
  - 조건 패턴 `(?(id)yes|no)`의 기본 사용법을 이해할 수 있다
- **핵심 키워드**: 명명 그룹, 비캡처 그룹, 역참조(패턴 내), `groupdict()`, `(?:...)`, `(?P<name>...)`, `(?P=name)`, `\1`, 조건 패턴, 캡처 그룹(복습)
- **도입 개념**: 비캡처 그룹(`(?:...)`), 명명 그룹(`(?P<name>...)`), 패턴 내 역참조(`\1`, `\2`, `(?P=name)`), `groupdict()`, 조건 패턴(`(?(id)yes|no)`)
- **선수 지식**: Chapter 6 (캡처 그룹 기초, group(n), groups()), Chapter 4 (수량자와 반복)
- **난이도**: ⭐⭐⭐
- **예상 학습 시간**: 3.5시간
- **Section 목록**:
  - 8.1 비캡처 그룹: `(?:...)` — 그룹핑만 필요하고 캡처는 불필요한 경우, `findall()` 결과에 미치는 영향

### Chapter 9: 전후방 탐색 (Lookaround)

- **학습 목표**:
  - 전방 탐색(lookahead)과 후방 탐색(lookbehind)의 개념과 차이를 설명할 수 있다
  - 긍정 전방 탐색(`(?=...)`)과 부정 전방 탐색(`(?!...)`)을 활용한 패턴을 작성할 수 있다

  - 전후방 탐색이 "소비하지 않는 매칭(zero-width assertion)"임을 이해하고, 이를 활용할 수 있다
- **핵심 키워드**: 전방 탐색(Lookahead), 후방 탐색(Lookbehind), 긍정/부정, `(?=...)`, `(?!...)`, `(?<=...)`, `(?<!...)`, 제로 너비 단언(Zero-width assertion)
- **도입 개념**: 전방 탐색(Lookahead), 후방 탐색(Lookbehind), 긍정 전방 탐색(`(?=...)`), 부정 전방 탐색(`(?!...)`), 긍정 후방 탐색(`(?<=...)`), 부정 후방 탐색(`(?<!...)`), 제로 너비 단언(Zero-width assertion)
- **선수 지식**: Chapter 8 (고급 그룹핑, 비캡처 그룹 문법)
- **난이도**: ⭐⭐⭐⭐
- **예상 학습 시간**: 3시간
- **Section 목록**:

  - 9.2 긍정 전방 탐색: `(?=...)` — 특정 패턴이 뒤따르는 위치 매칭
  - 9.3 부정 전방 탐색: `(?!...)` — 특정 패턴이 뒤따르지 않는 위치 매칭

## 4.3 효과적인 학습 전략

1. **실습 중심 학습**: 모든 패턴은 반드시 직접 작성하고 실행해보세요. [regex101.com](https://regex101.com)에서 Python 모드로 실시간 테스트하는 습관을 들이면 학습 효율이 크게 높아집니다.

2. **점진적 패턴 구성**: 복잡한 패턴을 한 번에 작성하려 하지 말고, 작은 부분부터 시작하여 점진적으로 확장하세요. 각 단계마다 테스트하는 습관이 중요합니다.

3. **패턴 읽기 연습**: 다른 사람이 작성한 정규표현식을 분석하는 연습을 병행하세요. Stack Overflow나 GitHub에서 실제 사용되는 정규표현식을 찾아 해석해보는 것이 좋습니다.

4. **오답 노트**: 매칭이 예상과 다르게 동작한 경우를 기록해두세요. 특히 탐욕적/비탐욕적 매칭, 앵커, 그룹핑에서 자주 실수가 발생합니다.

5. **반복 학습**: Part 1~2를 마친 후 Part 3로 넘어가기 전에, Part 1~2의 내용을 활용한 간단한 프로젝트(텍스트 파일에서 특정 정보 추출 등)를 수행하면 기초가 탄탄해집니다.
